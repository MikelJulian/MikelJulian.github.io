Batch Export to 0,0,0 for 3ds Max

This tool automates the process of exporting multiple assets from a 3ds Max scene by following an intelligent workflow:

  • It creates temporary duplicates of all visible assets in the scene.
  • These duplicates are placed at the world origin (0,0,0) without altering the original hierarchy or transforms.
  • Each asset is then exported individually with the name of the current Max file.
  • After export, the duplicates are removed, keeping your original scene untouched and ordered.

This workflow ensures precision exports while preserving the original scene layout, especially useful for game engines and batch import pipelines.
